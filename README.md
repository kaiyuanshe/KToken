# KToken
基于区块链的社区贡献激励方案

## 重启计划路线图
在以太坊 http://ethereum.org/ 测试网上建立 KToken 贡献积分（已完成）

建立第一版测试合约。（已完成）

完成使用指导手册。（已完成）

制定贡献积分获得规则。

KToken 贡献积分在内部分发使用。
媒体组
开放黑客松项目组
OpenDigger

收集反馈，更新规则。（有可能需要迭代几个测试版本）

推广 2-3 个合作社区使用 KCoin 积分或可实现兑换。

修正规则以及兑换方式。

修正规则运行推广 5 个具有代表性的社区测试运作。

完成最终版本并进行代码审计。

订正代码并发布上主网。

映射测试网地址积分。
